Vue E-Commerce Product Display

Aplikasi frontend berbasis Vue.js yang menampilkan katalog produk e-commerce dengan mengambil data dari FakeStoreAPI. Produk ditampilkan berdasar kategori seperti Men’s Clothing, Women’s Clothing, elektronik, dan lainnya. Terdapat loading indicator saat data sedang dimuat.

Vue 3

Framework Frontend

FakeStoreAPI

API Data Produk
Lihat di GitHub

Technologies Used

Vue.js HTML5 CSS3 JavaScript

⭐ Key Features

  • Mengambil produk dari FakeStoreAPI.
  • Display produk kategori seperti pakaian dan elektronik.
  • Indikator loading saat fetch data.
  • Responsive UI untuk tampilan produk.
  • Dikembangkan dengan Vue.js 3 dan struktur komponen.

Presensi Karyawan

Sistem Presensi Karyawan berbasis web yang dikembangkan untuk membantu proses pencatatan kehadiran karyawan secara digital. Aplikasi ini memungkinkan karyawan melakukan presensi masuk dan keluar, serta memudahkan admin dalam mengelola data kehadiran, data karyawan, dan laporan presensi secara terstruktur dan efisien.

⭐ Key Features

  • Sistem login multi-role (Admin & Karyawan).
  • Presensi masuk dan keluar berbasis waktu.
  • Manajemen data karyawan oleh admin.
  • Rekap dan laporan data presensi.
  • Penyimpanan data terintegrasi dengan database MySQL.

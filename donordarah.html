E-commerce Catalog

E-commerce Catalog merupakan aplikasi frontend berbasis Vue.js yang menampilkan daftar produk e-commerce secara dinamis. Data produk diambil dari FakeStoreAPI dan ditampilkan dalam bentuk katalog interaktif yang responsif. Aplikasi ini dirancang untuk melatih pemahaman penggunaan API, komponen Vue, serta konsep Single Page Application (SPA).

⭐ Key Features

  • Menampilkan daftar produk dari FakeStoreAPI.
  • Halaman detail produk tanpa reload (SPA).
  • Penggunaan komponen Vue.js (reusable).
  • Fetching data API menggunakan JavaScript.
  • Tampilan responsif untuk berbagai ukuran layar.
  • Routing halaman menggunakan Vue Router.
